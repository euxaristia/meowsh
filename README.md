# meowsh

A POSIX-compliant shell implementation.

## Features

- POSIX compliance (WIP)
- Built-in commands (alias, jobs, etc.)
- Line editing support
- Job control
- Variable expansion
- Redirections

## Building

To build `meowsh`, run:

```bash
make
```

## Usage

After building, you can start the shell with:

```bash
./meowsh
```

- `src/`: Source code
